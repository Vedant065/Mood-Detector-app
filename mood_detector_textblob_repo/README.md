# Mood Detector (TextBlob)

Simple Streamlit app that analyzes text sentiment using TextBlob (no NLTK data required).

## Run locally

1. Create & activate a virtual environment (recommended).
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   streamlit run mood_detector.py

## Deploy to Streamlit Cloud

Upload these files to a GitHub repo and create a new app on https://share.streamlit.io with main file `mood_detector.py`.
